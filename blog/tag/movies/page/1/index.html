Fairly One-Dimensional

Sentiment analysis of movie taglines

2015-06-16

I recently came across Randal Olson's excellent post on how movie rating reasons have changed over time, and the fantastic data available via IMDb's interfaces. Among the many interesting data available here are movie taglines. I'd been looking for an excuse to try out some rudimentary sentiment analysis, and these tailgates brought to mind an immediate question: have movie taglines gotten more negative over time?

Getting a "sentiment score"

Tools

I extracted all of the taglines along with each film's genre (using the taglines.list and genres.list files) into a pandas data frame. I used the excellent Natural Language Toolkit (NLTK) as a scaffolding for all of these analyses. To limit the analysis to movies with taglines in English, I used langdetect. Finally, I used the SentiWordNet 3.0 (SWN) database to get the actual scores for each word. The final analysis was performed on 7,381 taglines.

The problem of multiple meanings

One of the challenges of sentiment analysis I had underestimated before attempting this analysis is that words can have dramatically different connotations depending on their context. For example, "killing" might generally have a strong negative valence, but in the context of "killing it," might be positive. Moreover, each word is represented in the SWN database separately for each of its unique contextual meanings.

While I made rudimentary attempts to deal with this problem, I mostly ignored it because the absolute sentiment score of a given tagline is irrelevant in these analyses. All that matters here is the relative sentiment scores across years/genres, so, hopefully, noise due to multiple meanings should average out. That said, the multiple meanings problem is absolutely a confound in these analyses.

Putting it together

First, I tokenized each tagline and extracted the parts of speech of each word. I then compared nouns, verbs, adjectives, and adverbs to the SWN database to extract each word's net sentiment score (a word's positive score minus its negative score)for matches in the database with the same part of speech. To get a final sentiment score for a tagline, I simply summed each word's sentiment scores.

Because sentiment scores are represented as the difference between positive and negative, a value of 0 indicates a neutral connotation, and positive/negative values represent positive/negative connotations.

Results

So, what happened? Have taglines gotten more negative over the last 50 years?

Unfortunately, it seems the answer is no. In fact, it seems they've remained mostly unchanged. The extreme variation between 1950 and 1975 is mostly due to a decreased number of movies with taglines in the IMDb database for those years.
